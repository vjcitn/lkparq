# Extract UTR and codon features from a GTFParquet object

<div class="ref-description section level2">

Functions to extract UTR (untranslated region) and codon features from a
GTFParquet object. These features are stored in the features.parquet
file generated by gtf\_to\_parquet.py.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
utrs(x, type = "both", filter = NULL)

# S4 method for class 'GTFParquet'
utrs(x, type = c("both", "5prime", "3prime"), filter = NULL)

codons(x, type = "both", filter = NULL)

# S4 method for class 'GTFParquet'
codons(x, type = c("both", "start", "stop"), filter = NULL)
```

</div>

</div>

<div class="section level2">

## Arguments

-   x:

    A `GTFParquet` object.

-   type:

    For `utrs`: one of `"both"`, `"5prime"`, or `"3prime"`. For
    `codons`: one of `"both"`, `"start"`, or `"stop"`.

-   filter:

    Optional named list for filtering features.

</div>

<div class="section level2">

## Value

A [GRanges](https://rdrr.io/pkg/GenomicRanges/man/GRanges-class.html)
object with metadata columns including `feature_type`, `transcript_id`,
and `gene_id`.

</div>

<div class="section level2">

## Examples

<div class="sourceCode">

``` r
if (FALSE) { # \dontrun{
gtf <- GTFParquet(system.file("gc49", package="lkparq"))

# 5' UTRs
utr5 <- utrs(gtf, type = "5prime")

# 3' UTRs
utr3 <- utrs(gtf, type = "3prime")

# Start codons
start <- codons(gtf, type = "start")

# Stop codons
stop <- codons(gtf, type = "stop")
} # }
```
