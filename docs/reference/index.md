# Package index

<div class="section level2">

## All functions

</div>

<div class="section level2">

-   `genome(<GTFParquet>)` `seqinfo(<GTFParquet>)` : GTFParquet class
-   `GTFParquet()` : Create a GTFParquet object
-   `gene_types()` `transcript_types()` : List available gene or
    transcript types
-   `protein_coding_genes()` `lncRNA_genes()` : Convenience functions
    for common gene types
-   `genes(<GTFParquet>)` `transcripts(<GTFParquet>)`
    `exons(<GTFParquet>)` `cds(<GTFParquet>)` : Extract genomic features
    from a GTFParquet object
-   `gtf_metadata()` : Get GTF metadata
-   `genes_in_region()` `transcripts_in_region()` : Find features
    overlapping a genomic region
-   `transcriptsBy(<GTFParquet>)` `exonsBy(<GTFParquet>)`
    `cdsBy(<GTFParquet>)` : Extract and group genomic features from a
    GTFParquet object
-   `utrs()` `codons()` : Extract UTR and codon features from a
    GTFParquet object

</div>
